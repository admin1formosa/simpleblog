bitbucket同步

Date 週二 07 一月 2014 Tags bitbucket

針對在 OpenShift 上的 Python 3.3 網站程式碼, 設法與 https://2013falla@bitbucket.org/2013falla/2013c1project.git 同步

cd v:\c1project

git remote add 2013c1project https://2013falla@bitbucket.org/2013falla/2013c1project.git

git push -u 2013c1project --all # pushes up the repo and its refs for the first time

git push -u 2013c1project --tags # pushes up any tags

取得本網站的程式碼: git clone https://2013falla@bitbucket.org/2013falla/2013c1project.git

一旦針對原先為 OpenShift Git repository 建立的專案程式碼, 進行上述處理後, git remote -v 會將 2013c1project 放在首位, 若隨後執行:

git add .

git commit -m "my commit note"

git push

之後, 將會內定將版本快照送到 2013c1project, 也就是 Bitbucket repository, 若要將同一 commit 送到 OpenShift, 則需要:

git push origin
