virtualbox簡單使用紀錄

Date 週二 07 一月 2014 Tags virtualbox

虛擬機軟體最出名的有兩家 1. virtualbox - 免費，在被 oracle 收購前就已開放) 2. vmware - 部分收費

功能上來說 vmware 強過 virtualbox 很多，但本次只做簡單的工作...so....

這次因為學校的 creo server 過於老舊導致硬碟掛點，所以打算使用 virtualbox 來做虛擬 server 認證

步驟

  1. download virtualbox
  2. install virtualbox and drive(包括顯卡 網卡 等)
  3. 準備 OS 安裝光碟 or iso 等其他映像檔案
  4. 開啟 virtualbox 確定其 網卡已成功安裝置 host OS
  5. 在 virtualbox 選擇 OS 類型，並安裝
  6. 安裝完成，關閉 virtualbox， 設定與 host 共用光碟機
  7. 網路調整成僅限主機介面卡(僅在自己電腦認證)，如需暴露在外則要改成橋接(須加裝一網卡)，mac 網卡位置 需改成與原本 creo 綁定的 mac address 相同
  8. 在 virtualbox 內安裝認證server
  9. 確定在 host 能夠 telnet virtualbox 內的 server port(7788)
  10. 在 virtualbox 內確定 server 服務有確實啟動
  11. 將在host 的 creo 認證主機指向 virtualbox 的虛擬環境 內部IP
